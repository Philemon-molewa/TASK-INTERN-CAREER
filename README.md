Overview.
The Online Examination System is a Java desktop application that allows users to register, log in, and access exam-related resources. The project demonstrates Java GUI development using Swing, MySQL database integration, and basic user authentication.

Features.
User Registration – Add new users to the system.
User Login – Secure authentication for registered users.
Database Integration – Store and retrieve user data from MySQL.

Technologies Used.
Java SE Swing for GUI
MySQL database backend
MySQL Connector/J JDBC driver
NetBeans IDE development environment

How to Run the Project.
Set Up Database – Start MySQL, import the onlineexam database.
from NetBeans Load the project and add the MySQL Connector/J library.
Update the database credentials in the project like the password and root.
Run Application.
